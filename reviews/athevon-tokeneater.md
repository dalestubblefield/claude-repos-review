## Summary

Native macOS menu bar app monitoring Claude AI usage limits in real-time. Swift 5.9 + SwiftUI + WidgetKit. Floating overlay for active Claude Code sessions, customizable themes, smart pacing indicators. Protocol-based DI, separated app host / widget extension / shared services. Requires Pro or Team tier.

## Would / Would Not

**Would:** Native macOS menu bar = zero-friction ambient awareness of token burn rate. WidgetKit dashboard is polished UX vs. CLI tools.

**Would not:** Swift/macOS only. Requires Pro/Team tier. Overlaps with ccstatusline and ccusage — pick one monitoring approach.
