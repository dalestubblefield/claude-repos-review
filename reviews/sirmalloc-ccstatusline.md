## Summary

Customizable status line formatter for Claude Code CLI. Real-time model info, git branch, token usage, session duration. TypeScript + React/Ink, Node.js and Bun. Modular widget architecture, Powerline-style rendering, zero-config defaults.

## Would / Would Not

**Would:** Zero-friction quality-of-life upgrade for daily Claude Code sessions — modular widget system means you only use what you need.

**Would not:** Two ccstatusline forks in the starred list (this + pcvelz fork) — evaluate which has better API usage tracking before committing to one.
