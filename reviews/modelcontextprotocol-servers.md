## Summary

Official reference implementations of MCP servers from Anthropic. Language-agnostic — SDKs in TypeScript, Python, Go, Rust, Java, C#, PHP, Ruby, Swift, Kotlin. Core servers: Everything, Fetch, Filesystem, Git, Memory, Sequential Thinking, Time. Educational examples, not production-ready. Community integrations via MCP Registry.

## Would / Would Not

**Would:** Authoritative reference for building MCP servers — canonical starting point and the pattern every other MCP tool builds on.

**Would not:** Educational implementations only — production use requires security hardening. Actual useful community servers are now in the external MCP Registry, not this repo.
